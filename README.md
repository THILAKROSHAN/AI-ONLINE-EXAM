# AI Examination Portal

A professional Online AI Examination Portal suitable for schools, colleges, coaching institutes, universities, and organizations.

## Features

- **Multi-Organization Support**: Complete data isolation between organizations
- **Role-Based Authentication**: Super Admin, Organization Admin, Sub Admin, Student
- **Student Management**: CRUD, Import/Export, Profile Photos, Auto-generate credentials
- **Subject Management**: Complete CRUD with department, semester, year tracking
- **Question Bank**: Multiple question types, bulk import/export, difficulty levels
- **AI Question Generator**: Powered by Google Gemini AI
- **Exam Management**: Timer, auto-submit, resume support, security features
- **Student Portal**: Dashboard, exams, results, performance analytics
- **Results & Analytics**: Auto-evaluation, manual grading, PDF reports
- **Reports**: PDF, Excel, CSV exports
- **Audit Logs**: Complete activity tracking
- **Email Notifications**: Credentials, exam reminders, results
- **Dark/Light Mode**: Modern responsive UI

## Technology Stack

- **Frontend**: React 19 + Vite + Tailwind CSS
- **Backend**: Firebase (Auth, Firestore, Storage, Cloud Functions)
- **AI**: Google Gemini AI API
- **Charts**: Recharts
- **Reports**: jsPDF, XLSX

## Prerequisites

- Node.js 18+
- npm 9+
- Firebase CLI
- Google Gemini AI API Key
